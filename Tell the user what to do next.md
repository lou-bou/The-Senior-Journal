Experience - I was using an e-commerce shipping company's website, and when I wanted to download the sheet file for the orders, it wouldn't do so. Turns out, I had to wait a little bit for the file to initialize downloading, but there was no UI / text to tell me to wait.

Lesson - Always tell the user what to do after each action. More specifically, tell them to wait when they need to wait so they don't think there's a problem with the software.
